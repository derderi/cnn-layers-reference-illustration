CNN Layers Reference Guide

Bilingual reference guide for understanding different layers in Convolutional Neural Networks (CNN).

By: Dr.Eldirdiri Fadol Ibrahim Fadol

Layer Definition Function Illustration
Convolutional Layer هذه الطبقة تنشئ نواة التلافيف التي تُطبق على بيانات الإدخال على بُعد مكاني أو زمني واحد لإنتاج مصفوفة المخرجات. إذا كانت use_bias صحيحة، يتم إنشاء متجه الانحياز وإضافته إلى المخرجات. وأخيرًا، إذا لم يكن activation فارغًا، يتم تطبيقه على المخرجات أيضًا. Filter/Kernel: الفلتر أو النواة هو مصفوفة الوزن، التي تكون أصغر حجماً من الإدخال. تنتقل على بيانات الإدخال لاكتشاف أنماط أو ميزات معينة. Input Image -> [Filter/Kernel] -> Feature Map
Activation Layer طبقة التنشيط تطبق وظيفة تنشيط لإدخال عدم الخطية في النموذج، مما يسمح له بتعلم أنماط معقدة. Main Functions: إدخال عنصر غير خطي في المعادلة وتحديد نطاق المخرجات. (from Convolutional Layer) -> [Activation Function (e.g., ReLU)] -> Activated Output
Batch Normalization Layer طبقة تطبيع الدفعة تقوم بتطبيع مخرجات طبقة التنشيط السابقة عن طريق تعديل وتوسيع التنشيطات. Momentum Parameter: يحدد مدى تحرك المتوسط والانحراف المعياري بالنسبة للتكرار السابق، مما يضمن تدريبًا مستقرًا وأسرع. (from Activation Layer) -> [Batch Normalization] -> Normalized Output
Pooling Layer طبقة التجميع تقلل الأبعاد المكانية (العرض والارتفاع) للإدخال، مما يساعد في تقليل عدد المعلمات والحسابات في الشبكة. Commonly Used Methods: MaxPooling و AveragePooling هما أكثر طرق التجميع استخدامًا. (from Convolutional Layer) -> [Pooling (MaxPooling/AveragePooling)] -> Pooled Output
Fully Connected Layer طبقة متصلة بالكامل تربط كل خلية عصبية في طبقة بخلية عصبية في طبقة أخرى، وتُستخدم عادةً كطبقة نهائية في الشبكة العصبية التلافيفية. Main Function: تُستخدم كطبقة نهائية لتحديد الخصائص التي تتناسب بشكل أفضل مع كل فئة معينة. (from previous layers) -> [Fully Connected Layer] -> Output (Class Scores) 
8. activation (التنشيط): وظيفة التنشيط لإستخدامها. إذا لم يتم تحديدها، لن يتم تطبيق أي تنشيط.
9. use_bias (استخدام التحيز): منطقية، إذا كانت True، سيتم استخدام متجه التحيز.
10. kernel_initializer (مهيئ النواة): مهيئ لتطوير النواة.
11. bias_initializer (مهيئ التحيز): مهيئ لتطوير التحيز.
12. kernel_regularizer (منظم النواة): وظيفة منظم مطبقة على النواة.
13. bias_regularizer (منظم التحيز): وظيفة منظم مطبقة على التحيز.
14. activity_regularizer (منظم النشاط): وظيفة منظم مطبقة على المخرجات.
15. kernel_constraint (قيد النواة): وظيفة قيد مطبقة على النواة.
16. bias_constraint (قيد التحيز): وظيفة قيد مطبقة على التحيز.
